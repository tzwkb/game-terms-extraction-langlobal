# 术语标注助手 v3.0

游戏术语提取工具的配套**人工审校工作台**：AI 提取的候选术语在这里完成人工标注、审核、修订，导出审定术语表。

单 HTML 文件，浏览器双击打开即用，无需安装。数据全程在本地浏览器内，不上传。

> 零外部依赖：表格解析库（SheetJS 0.20.2）已内嵌文件内，完全离线可用。

## 与提取工具的闭环

```
原文.xlsx ──→ [提取工具] ──→ 候选术语_模板.xlsx（未审核）
                                    │ ② 导入合并
原文.xlsx ──① 导入──────────→ [本工具] ──③ 标注/审核──→ 术语表_*.xlsx（8列模板）
                                                              │ ④ 回流作下次提取的术语表
                                                              ▼
                                                        [提取工具]
```

## 快速上手

1. **导入原文 Excel**，在工具栏指定 Key 列 / 原文列 / 译文列；
2. **导入术语表**（提取工具产出的 `候选术语_模板.xlsx`，或任何含「术语原文」列的表），选择合并策略：
   - 跳过已有 —— 保留本地修改（默认）
   - 仅补空白字段 —— 合并 AI 增量结果推荐
   - 覆盖已有
3. 标注与审核（见下方快捷键）；
4. **导出术语表 (8列模板)** —— 缺失的 Key值/来源原文 按首次出现行自动回填。

## 添加术语的五种方式

| 方式 | 操作 | 适用场景 |
|---|---|---|
| 划词添加 | 选中原文列任意文字，按 <kbd>A</kbd> | 词条是句子的一部分 |
| 整行添加 | 行首 ➕ 按钮 | 原文列整格就是一个词条（技能表/物品表） |
| 批量添加 | <kbd>Q</kbd> 进勾选模式 + Shift 连选 + <kbd>Enter</kbd>，可统一设分类 | 成片的词条行 |
| 手动新增 | 右侧「➕ 新增术语」按钮，直接填 原文/分类/译文/备注 | 原文里没有的词、或未导入原文时直接录入 |
| 导入合并 | 「导入术语表」读任何含「术语原文」列的 Excel，三种冲突策略 | 合并 AI 候选 / 同事的表 |

任何方式添加后自动：状态=未审核、全文高亮即时生效、倒排索引登记（点击跳转）、来源行 Key 自动携带（手动新增的词导出时按首次出现行回填，原文中不存在则留空）；重复添加会被拦截。

## 快捷键与操作

| 操作 | 方式 |
|---|---|
| 划词加术语 | 选中原文列文本，按 <kbd>A</kbd> |
| 删除 / 编辑选中术语 | <kbd>D</kbd> / <kbd>E</kbd> |
| 整行加术语 | 行首 ➕ |
| 批量加术语 | <kbd>Q</kbd> 进入勾选模式，Shift 连选，<kbd>Enter</kbd> 批量添加（可统一分类） |
| 审核状态切换 | 术语列表 ✓（黄=未审核，蓝=已审核，绿=译文命中；匹配大小写不敏感） |
| 跳转定位 | 单击右侧术语循环跳转匹配行；顶部行号跳转 |
| 查看整行全文 | 双击行号（超长单元格列表内只显示前 600 字） |
| 编辑高亮术语 | 双击正文中的高亮词 |
| 保存项目 | <kbd>Ctrl+S</kbd>（JSON 工程，含原文+术语+列配置，加载即复原） |

筛选：Key/原文/译文列支持 包含/不包含/等于（译文列另有 为空/不为空，查漏译）；列显隐可勾选。

## 数据安全

- **自动备份**：默认每 5 分钟下载一份工程 JSON（可调/关）；
- **崩溃草稿**：术语库每分钟存入浏览器本地，崩溃重开提示恢复（仅术语，原文需重新导入；超约 3 万词时因浏览器配额自动停用）。

## 8 列模板（与提取工具的接口约定）

`Key值 / 术语分类 / 术语原文 / 术语译文 / 备注 / 来源原文 / 审核状态 / 最新修订时间`

导入只要求「术语原文」列存在，常见别名（ukey、译文、分类等）自动识别。

## 容量

10 万行 × 3 万术语全程无感（虚拟滚动 + AC 自动机高亮 + 倒排索引）。行数可至 xlsx 格式天花板 104.8 万（滚动高度映射，全浏览器安全）；术语量任意规模不冻结界面（词典分片后台构建）。崩溃草稿兜底限约 3 万词。详见 `docs/标注工具性能改造方案_20260612.md`。
